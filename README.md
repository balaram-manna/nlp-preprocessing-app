# nlp-preprocessing-app



\# NLP Preprocessing Streamlit App



This is a Streamlit web app for NLP preprocessing.



\## Features

\- Tokenization

\- Text Cleaning

\- Stemming

\- Lemmatization

\- Bag of Words



\## How to Run



1\. Install dependencies:

&nbsp;  pip install -r requirements.txt



2\. Run the app:

&nbsp;  streamlit run app.py
